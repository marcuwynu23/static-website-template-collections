# Static Website Collections

A curated collection of static websites built for learning, experimentation, and quick deployment.

## Overview

This repository contains multiple static website projects, each showcasing different design styles, layouts, and web technologies. It's perfect for inspiration, practice, or using templates for personal projects.

## Features

- Collection of ready-to-use static websites
- Organized by design type and complexity
- Fully responsive designs
- Minimal setup required
- Built with HTML, CSS, and JavaScript

## Getting Started

1. **Clone the repository:**

```bash
git clone git@github.com:USERNAME/static-website-collections.git
```

2. **Open a website:**
   Navigate to the folder of the desired website and open `index.html` in your browser.
